JavaScript
자바스크립트는 사용자와 상호작용을 하는 언어이다.
웹브라우저는 한번 화면에 출력되면 자기자신을 바꿀 수 없다.
script 태그 안에는 JavaScript가 오도록 약속되어 있다.
JavaScript Data Type (자료형) :
Boolean, Null, Undefined, Number, String, Symbol / Object
  • Boolean : True, False
  • Number : 숫자
  • String : 문자열
  • Object : 객체
  • 조건문 Conditional Statements
    조건에 다라서 다른 순서의 기능들이 실행.
    if(){}else{}
    if 뒤 () 안에는 불리언 데이터 타입이 옴 (True / False)
  • while 반복문
    while() 괄호 안에는 불리언 데이터 타입이 옴
  • 변수 Variable : 바뀔 수 있는 어떤 값
  • 배열 Array []
    → var Boolean = ['true', 'false']
    배열에 있는 각각의 항목들을 원소element라고 함
    정보가 순서대로 저장 됨
  • 함수 Function, Method(메소드는 객체에 속한 함수)
    입력 (parameter매개변수, argument인자)과 출력(return)으로 되어 있음
객체 Object {}: 서로 연관된 함수와, 서로 연관된 변수를 같은 이름으로 그룹핑해서 잘 정리정돈하기 위한 도구
객체에 소속된 변수 = property
script 태그 안의 코드들을 js 파일로 옮기고, script 코드에 src="파일명.js" 를 적어서 코드를 적용할 수 있다.
